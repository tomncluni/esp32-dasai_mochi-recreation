# esp32-dasai_mochi-recreation
A set of code preloaded with animations to replicate a Dasai Mochi with an esp32 and an OLED screen.

The [Dasai Mochi](https://dasai.com.au) is a small desktop/dashboard buddy. 
This code aims to recreate this product with an esp32 and an OLED screen.
There are two versions of the code available, one which loops the idle animation until a button is pressed, where it then plays a random animation before returning to the idle state. The other version does this automatically on a timer.

Inspired by the work of [upiir](https://github.com/upiir/esp32s3_oled_dasai_mochi) and utilising animations from [watcher-mochi](https://github.com/pham-tuan-binh/watcher-mochi)
